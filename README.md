# Bitmap Font Editor
Can Open, Save, and create Microsoft BitMap FON and Raw FNT (ROM) files. 
Convert any True Type Font (TTF) to FON or FNT format.
New formats include Deluxe Paint II Fonts, Amiga Fonts, Veridian APL Fonts, and TEGL Fonts. Plus a QBasic script interpreter!
